Generic outline UI glyph used across the system — chevrons on buttons, mail/phone/pin in contact blocks, methodology icons (target, bar-chart, trending-up, users, search).

```jsx
<Icon name="chevron-right" size={14} color="#fff" />
```

Variants: any of the 13 names in Icon.d.ts. Methodology blocks use size 80 and `color="var(--color-primary)"`. Never use Icon for Facebook/Instagram/LinkedIn/TikTok marks — use SocialIcon instead.
